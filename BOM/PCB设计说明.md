## 二、对乙方的技术要求（**必须照做的**）

### 2.1 板级约束

| 项目 | 要求 | 原因 |
|---|---|---|
| **PCB 层数** | 双层板（2 层）即可 | 满足设计 + 控制成本 |
| **板厚** | 标准 1.6mm | 嘉立创默认 |
| **板材** | FR-4 | 标准玻纤板 |
| **铜厚** | 内/外层 **2 oz**（70µm） | 大电流回路必需 |
| **最小线宽** | 6 mil（0.15mm） | 嘉立创免费打样最小值 |
| **最小线距** | 6 mil | 同上 |
| **过孔尺寸** | 0.3mm/0.6mm（孔/盘） | 嘉立创默认 |
| **阻焊颜色** | 不限（蓝/绿/黑都行） | 个人偏好 |
| **PCB 尺寸** | 不超过 10×10cm | 嘉立创免费打样上限 |

### 2.2 高优先级布局/布线规则（**必须严格遵守**）

#### A. 采样电阻 Rsns（10mΩ 2512）

```
要求：
1. 必须 4 端开尔文连接
   - 大电流端（强制端）走 2 oz 厚铜+多过孔
   - 信号端（传感端）从 Rsns 焊盘的 "另两个端点" 单独引出
2. SRP/SRN 信号必须差分对走线
   - 等长（误差 < 5mm）
   - 间距 0.2mm，宽度 0.2mm
   - 全程平行
3. 每颗 BQ 芯片（BQ76920、BQ34Z100）的 100Ω 滤波电阻
   必须独立！不能并联共用
4. 100nF 差分电容必须紧贴 BQ 芯片放置（< 5mm）
```

#### B. BQ76920 VC0~VC5 电池采样网络

```
要求：
1. RC 网络（100Ω + 1µF）
   - R 紧贴 BQ76920 引脚（< 3mm）
   - C 跨接相邻 VC 节点（不是对地！）
2. VC0~VC5 走线尽量短直
3. 严禁穿过开关电源、CAN、SPI 等高速信号区域
4. VC 走线建议靠近板边缘，不要从板中间穿
```

#### C. BQ34Z100-G1 电压翻译电路

```
要求：
1. 1.5MΩ + 68kΩ 分压电阻必须使用 1% 精度
2. 100nF 滤波电容紧贴 BQ34Z100 BAT 引脚
3. 不要在分压点附近放置高速信号
```

#### D. 大电流回路（电池负极 → MOS → PACK-）

```
要求：
1. 该回路铜皮宽度 ≥ 5mm（满载 10A 时）
2. 多过孔（≥4 个并联）减小阻抗
3. 走最短路径
4. MOS 管下方铺大块铜皮散热
```

#### E. 接地策略（**最关键的一项**）

```
要求：
1. 划分 4 种地：
   - PACK_GND（功率地，PACK- 端子侧）
   - BAT_NEG（电池组 B-，含 VSS、VC0）
   - AGND（模拟地，BQ 芯片 VSS、NTC 网络）
   - DGND（数字地，STM32、LCD、L610、蜂鸣器）

2. 连接策略（"星形单点接地"）：
   - PACK_GND 通过 MOS Source 与 BAT_NEG 隔离（MOS 关断时）
   - AGND = BAT_NEG，直接合并
   - DGND = AGND，但**只能在 Rsns 电池侧单点连接**
   - 严禁 DGND 大铜皮直接覆盖到电池采样区域

3. 大电流不能流过模拟地！
```

#### F. STM32 周边

```
要求：
1. 每个 VDD 引脚必须就近放置 100nF 退耦电容（< 3mm）
2. HSE 8MHz 晶振走线短，下方禁止铺其他信号
3. SWD（PA13/PA14）走线短直，禁止靠近 RF 信号
4. NRST 周围不要放噪声源
```

#### G. L610 4G 模组

```
要求：
1. VBAT 引脚必须有 100µF 电解电容（紧贴模组 < 10mm）
2. 4G 信号天线引出走 50Ω 阻抗匹配
3. UART 信号线远离 4G RF 区域
4. NET_STATUS LED 单独走线，不与其他信号并行
```

### 2.3 中优先级建议（可商量）

| 项目 | 建议 |
|---|---|
| 测试点 | I²C/SPI/UART 信号引出测试点（贴片焊盘 1.5×1.5mm） |
| 标识丝印 | 关键元件位号 + 关键端子标注（B+, B-, PACK+, PACK-, CAN_H, CAN_L 等）|
| LED 位置 | 放在板边缘，方便观察 |
| 按键位置 | 板边缘，预留手指操作空间 |
| 调试接口 | SWD 排针 + USB-TTL 排针放板边 |

### 2.4 低优先级（乙方自由发挥）

- 元件具体摆放位置（在不违反 2.2/2.3 的前提下）
- 走线美观度（曲线/直角自选）
- 装饰性丝印
- 板形是否要圆角

## 六、风险预警与应对

### 6.1 常见的"翻车"场景

| 风险 | 表现 | 应对 |
|---|---|---|
| **乙方用 AI 自动布线** | 走线杂乱、关键约束没遵守 | 在 2.2 节明确"必须人工布线" |
| **乙方甩货跑路** | 拿了定金后失联 | 30% 定金 + 第三方担保（淘宝订单/嘉立创社区交易）|
| **乙方画完不能改** | 给的源文件版本兼容性差 | 提前确认 EDA 版本 |
| **元件库错误** | 封装画错（如把 SOT-23 当 SOT-323），导致焊不上 | 验收时核对 BOM + 封装 |
| **打样后发现问题** | 第一版试错难免 | 预留 1 次免费修改机会 |

### 6.2 验收时必查的项目

打到样板/Gerber 后，**自己用嘉立创 EDA 打开核对**：

- [ ] 板尺寸 ≤ 10×10cm
- [ ] 双层板 2 oz 铜厚
- [ ] 关键元件位号是否齐全
- [ ] 电池端子、PACK 端子是否在板边方便接线
- [ ] LCD 接口位置是否合理
- [ ] SWD 调试口是否引出
- [ ] 测试点是否足够
- [ ] 丝印是否清晰
- [ ] DRC（设计规则检查）无错误
- [ ] Gerber 文件能在嘉立创下单系统正常预览
